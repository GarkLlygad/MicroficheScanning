To customize your own page
  1. Make a fork of the repo - Requires a github account and more info found here https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo
      - No need to worry about the cloning of the repo or anything, simply downloading your html and css page then changing locally and uploading the changed files will work fine! 
  3. Change the html/css as you desire
  4. Create a pull request to the main branch
All that I ask is that the id's are left alone on the page as I use them to anchor my automatic changes for the updating of card counts and rescans
